# Niall - You think your voice matters?

https://en.wikipedia.org/wiki/Neil

The project is named after a boy who believes his voice matters.

## Run the project

```shell
docker compose up -d --build
```

## Tech stack

- NestJS + Turborepo
- MongoDB
- Kafka
- Redis
- OpenTelemetry
- Loki + Prometheus + Tempo + Grafana
